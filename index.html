Duperrific Theme Engine by soska

The Duperrific Theme Egine is a real WordPress theme programming framework for theme authors.

What is the difference with the other Theme Frameworks?

Well, most of them are only simple themes with a lot of hooks, not a real programming framework. Duperrific Theme Engine is (or tries very hard to be) a real programming framework, that tries to enhance the experience of programming WP Themes using MVCish paradigms.
